## Copy a Dashboard
A quick and easy way to create a customized dashboard is to copy a defaulted one and edit it.

1. Navigate to **Dashboards** > Select **`Dashboard Name`**.
2. Click on {{< icon name="fa-cog" size="large" >}} > **Copy**.
