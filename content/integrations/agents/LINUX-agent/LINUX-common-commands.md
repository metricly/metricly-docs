## Linux Commands
 - `service netuitive-agent {stop|start|restart}`  
 - ``/etc/init.d/netuitive-agent {stop|start|restart}``  
 - `initctl {stop|start|restart} netuitive-agent`  
 - `systemctl {stop|start|restart} netuitive-agent`
